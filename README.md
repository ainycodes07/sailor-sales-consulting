# Sailor Sales Consulting

Sailor Sales Consulting is a modern consulting agency website built with HTML, CSS, and JavaScript. The project features a bold editorial layout, interactive UI elements, custom animations, and a fully responsive experience designed to showcase business consulting services with clarity and strong visual impact.

## Features

- Custom loading screen and animated cursor  
- Responsive layout for desktop and mobile  
- Scroll-based reveal animations  
- Interactive service and testimonial cards  
- Floating client cluster visuals  
- Styled comparison table and contact form  
- Hero-inspired footer design  
- Enhanced hover effects and motion-based interactions  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome  
- Google Fonts  

## Preview

A visually-driven consulting website focused on clean layouts, strong typography, motion interactions, and modern UI styling.
